# bili_gift

本插件旨在方便下载 b 站直播的礼物图标，包括 PNG 与 GIF 两种格式

## 更新

-   **2023.12.22**：
    -   修复了无法更新礼物的问题
-   **2023.2.8**：
    -   增加了一键下载新增礼物图标的功能
    -   现在可以对礼物名称进行模糊搜素
    -   缩小了 exe 文件大小
-   **2022.3.10**：
    -   礼物列表储存文件改为 礼物列表.json
    -   增加了对重复下载地址的识别
-   **2022.2.23**：
    -   修复了无法显示新礼物的问题
